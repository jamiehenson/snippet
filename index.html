Installation
The easy way is to install snippet withbower install snippet. Alternatively, download the package, and include snippet.css and snippet.js from/distin an appropriate place within your project. You're gonna need jQuery for this chap. Bower will take take of this, but if you don't use it then just a head's up.
Then...
Basic markup
It's very easy to set up one of these boxes. Just include snippets.css (or the supplied SASS version) and snippets.js in your page, and then whack down the following markup:
<div class="snippet-box">
  <div class="snippet-content" style="max-height: CUSTOM-BOX-HEIGHT-VALUE">
    CONTENT GOES HERE
  </div>
  <div class="snippet-expander"></div>
</div>
Append whatever styling classes you like to thesnippet-expanderelement. Play about with it above!
